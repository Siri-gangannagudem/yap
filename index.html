2. Printing Output

yap("Hello, world!");

3. Taking User Input

int age = spill();

4. Operations

Supports arithmetic, comparison, logical, and bitwise operations.

5. Conditional Statements

if (num > 0) {
    yap("Positive number");
} elif (num < 0) {
    yap("Negative number");
} else {
    yap("Zero");
}

6. Loops

for (int i = 0; i < 5; i = i + 1) {
    yap(i);
}

7. Arrays

int[] numbers = [1, 2, 3, 4];
yap(numbers[2]); // Outputs 3

8. Functions

def add(int a, int b) -> int {
    yeet a + b
}
yap(add(5, 3)); // Outputs 8

9. Stacks & Queues

// Stack
stack s;
s.stackPush(3);
yap(s.top());

// Queue
queue q;
q.queuePush(3);
yap(q.first());
